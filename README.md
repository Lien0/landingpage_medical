Se hace uso de las etiquetas meta: autor, description, robots, title, viewport y charset, se utilizan las etiquetas link para incrustar el archivo css asi como para
hacer la integracion de google fonts, se usa la etiqueta script de tipo modulo para integrar el archivo javascript principal y que sep euda hacer uso del export
en otros archivos javascript como es el caso de los mocks data archive que tengo, el otro script es de fontawesome.
Las fuentes usadas son Montserra para el texto en general y Noto Serif para titulos haciendo contraste entre ambos, estos se declaran en la parte del 
:root para la declaracion de variables en el archivo CSS, se usan arial label para los iconos y hacemos uso de altura y anchura minima de 44px para poder 
añadir la accesibilidad correspondiente a la landing page, se realiza la pagina con un pensamiento movil first y haciendo uso de media queries se
realiza el responsive para la adaptación a tablets y dispósitivos de escritorio.
Se crean animaciones con javascript para los contenedores del titulo y texto de cada sección, se consumen los datos para la creación de tarjetas que muestren
tanto las preguntas y respuestas frecuentes asi como los médicos disponibles, se les da una animación para un desplazamiento suave, se agrega un 
validador de nombre y correo validándolo dentro del javascript, asi como se agrega el uso de un smooth para el desplazamiento suave del scroll cuando se desplaza
con los botones de la barra del header y footer.
